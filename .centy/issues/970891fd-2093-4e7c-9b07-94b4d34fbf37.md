# Generalize context menu (Pin / Move / Duplicate) for all item types

Part of the IssuesList → GenericItemsList migration (#196).

## Goal

Move the right-click context menu (Pin, View, Move, Duplicate) out of `IssuesList` and into a generic system available to any item type.

## Tasks

- Create `useGenericContextMenu.ts` — generalizes `useIssueContextMenu` with item type awareness
- Create `GenericContextMenu.tsx` — renders context menu UI
- Update `MoveModal` and `DuplicateModal` to accept any item type (not just issues)
- Create `useGenericMoveActions.ts` — generalizes `useIssueMoveActions`
- Wire into `GenericItemsList`
