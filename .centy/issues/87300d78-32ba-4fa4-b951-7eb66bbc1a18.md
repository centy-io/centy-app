# Update Daemon Modal: Add restart requirement notice with CLI command and restart button

When the "Update Daemon" modal is shown, users need to know that after updating, they must restart the daemon for the changes to take effect.

## Requirements

- Add a notice/info message in the update daemon modal explaining that a restart is required after updating
- Display the CLI restart command so users can copy and run it manually:
  ```
  centy daemon restart
  ```
- Add a "Restart Daemon" button in the modal that triggers a daemon restart programmatically
- The restart button should ideally be shown after the update completes (or be available alongside the update action)

## Motivation

Currently the modal does not communicate that a restart is needed post-update, which can leave users confused when changes don't take effect immediately.
