# feat: sort organizations by A-Z and usage in organizations management page

## Feature Request

Add sorting capabilities to the organizations management page so users can better navigate and manage their tracked organizations.

## Desired Behavior

Users should be able to sort the organizations list by:

- **A-Z** – alphabetical order by organization name
- **Usage** – ordered by usage/spend (e.g., highest to lowest or lowest to highest)

## Use Case

As the number of tracked organizations grows, it becomes harder to find and prioritize them. Sorting by name helps locate specific orgs quickly, while sorting by usage helps identify top-spending or least-active organizations at a glance.

## Acceptance Criteria

- [ ] A sort control (dropdown or toggle) is visible on the organizations management page
- [ ] Sorting by A-Z orders organizations alphabetically by name
- [ ] Sorting by usage orders organizations by their usage/spend metric
- [ ] The selected sort order persists during the session (or optionally across sessions)
