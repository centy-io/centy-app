# Evaluate and migrate standalone workspace support

Part of the IssuesList → GenericItemsList migration (#196).

## Goal

Decide whether the "New Workspace" button and `StandaloneWorkspaceModal` are issues-only or should become a generic feature, then implement accordingly.

## Options

- **Issues-only:** Keep `StandaloneWorkspaceModal` as an injected slot into `GenericListHeader` via a prop
- **Generic:** Add workspace feature flag to `ItemTypeConfigProto` and wire into `GenericListHeader`

## Tasks

- Decide which approach based on product direction
- Implement the chosen approach
- Ensure the button/modal still works after the issues page switches to `GenericItemsList`
