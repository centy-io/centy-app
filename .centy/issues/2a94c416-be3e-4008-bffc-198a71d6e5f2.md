# Overhaul app design to e-ink style

Redesign the app's visual aesthetic to match an e-ink / digital paper style. This includes:

- High-contrast, near-black-and-white color palette with minimal use of color
- Sharp, clean typography optimized for low-color rendering
- Reduced gradients and shadows — favor flat, crisp borders and outlines
- Dithered or stippled textures where depth/shading is needed
- Muted, paper-like background tones (off-white, cream, light grey)
- Careful use of accent color (if any) — single, muted hue only
- UI components (buttons, cards, inputs) styled to feel printed/physical rather than glassy or material
- Motion/transitions kept minimal and purposeful — e-ink displays have slow refresh

Goal is a calm, focused, readable interface that feels like reading on paper.
