# Centralize color definitions into a single colors file

All color values should be defined in a single centralized file (e.g. `colors.ts`). `rgba(...)` and other color values are allowed, but only inside that file — the rest of the codebase should import named tokens from it.

**Goal:** Make colors easy to audit and keep consistent — not to eliminate rgba, but to ensure all color definitions live in one place.

**Tasks:**

- Identify the centralized color file (or create one if it doesn't exist)
- Grep the codebase for `rgba(` usages outside that file
- Replace each with a named token imported from the central file
