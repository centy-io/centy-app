# feat: untrack organization from organizations management page

## Feature Request

Add the ability to untrack (remove) an organization directly from the organizations management page.

## Use Case

Currently there is no clear way to remove a tracked organization from the management page. Users should be able to easily untrack orgs they no longer want to monitor.

## Acceptance Criteria

- [ ] Each organization row/card on the management page has an "Untrack" action (e.g., button or context menu item)
- [ ] Clicking "Untrack" prompts the user for confirmation before removing
- [ ] After confirming, the organization is removed from the tracked list and the UI updates accordingly
