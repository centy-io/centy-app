# Add Mermaid diagram rendering in item description

Currently, Mermaid diagram syntax inside item descriptions is rendered as raw text/code blocks instead of visual diagrams. We should integrate a Mermaid library to properly parse and render these diagrams inline.

## Problem

When a user includes a Mermaid diagram (e.g., flowcharts, sequence diagrams, ER diagrams) in an item's description using a fenced code block (```mermaid), it is displayed as plain text rather than a rendered visual diagram.

## Goal

Integrate `mermaid.js` (or equivalent) to detect Mermaid fenced code blocks in item descriptions and render them as SVG diagrams.

## Acceptance Criteria

- Mermaid fenced code blocks in item descriptions are rendered as visual SVG diagrams
- Fallback gracefully to a raw code block if rendering fails
- Library is lazy-loaded to avoid impacting page load performance
