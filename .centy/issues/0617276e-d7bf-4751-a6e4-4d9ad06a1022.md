# Add table view mode to GenericItemsList

Part of the IssuesList → GenericItemsList migration (#196).

## Goal

Add an optional table display mode to `GenericItemsList` so issues (and other item types) can render as a sortable data table instead of the card grid.

## Tasks

- Add a `view: "table" | "grid"` field to `ItemTypeConfigProto` (or derive from feature flags)
- Create `GenericItemsTable.tsx` — Tanstack Table wrapper, column-agnostic
- Create `useGenericItemsTable.ts` — builds columns from item type config (standard: #, title, status; optional: priority, dates via feature flags)
- Plug into `GenericListContent.tsx` as an alternative to `ItemsGrid` based on config
